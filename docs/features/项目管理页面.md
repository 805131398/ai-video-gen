# 项目管理页面功能

## 概述

将创建项目的功能整合到左侧菜单的"项目管理"页面中，提供统一的项目管理入口。

## 功能特性

### 1. 页面布局

- **页面标题**: "项目管理" - 使用 Poppins 字体，专业现代
- **副标题**: "管理您的所有视频创作项目" - 使用 Open Sans 字体
- **创建按钮**: 醒目的橙色 CTA 按钮，位于页面顶部第一位置
- **项目列表**: 响应式网格布局（1/2/3 列）

### 2. 项目卡片

每个项目卡片显示：
- **项目标题/主题**: 主要标识信息
- **项目状态**: 彩色徽章显示（草稿/进行中/已完成/已归档）
- **创建时间**: 格式化的日期显示
- **更新时间**: 最后修改时间
- **操作按钮**:
  - 查看详情（蓝色主按钮）
  - 删除项目（红色图标按钮）

### 3. 交互设计

- **Hover 效果**: 卡片悬停时阴影提升，标题颜色变化
- **平滑过渡**: 200ms 的颜色和阴影过渡动画
- **Cursor 指示**: 所有可点击元素添加 `cursor-pointer`
- **确认删除**: 删除操作需要用户确认

### 4. 空状态处理

当没有项目时显示：
- 文件夹图标
- 提示文字："还没有项目"
- 快速创建链接

## 设计规范

### 配色方案（SaaS 专业配色）

```css
Primary: #2563EB (信任蓝)
CTA: #F97316 (橙色强调)
Background: #F8FAFC (浅灰背景)
Text: #1E293B (深色文本)
Border: #E2E8F0 (边框灰)
```

### 字体配对

- **标题字体**: Poppins (现代、专业、几何感)
- **正文字体**: Open Sans (易读性强、人文主义)

### 状态徽章配色

- **草稿**: 灰色 (`bg-gray-100 text-gray-700`)
- **进行中**: 蓝色 (`bg-blue-100 text-blue-700`)
- **已完成**: 绿色 (`bg-green-100 text-green-700`)
- **已归档**: 深灰 (`bg-slate-100 text-slate-600`)

```

├── pages/

├── components/layout/
│   └── NavMenu.tsx              # 更新：添加项目管理菜单项

```

### 路由配置

- **路径**: `/projects`
- **组件**: `ProjectManagement`
- **菜单图标**: `FolderKanban` (来自 lucide-react)

### API 集成

使用现有的项目服务：
- `getProjects()`: 获取项目列表
- `deleteProject(id)`: 删除项目
- `createProject(data)`: 创建新项目（通过导航到 `/projects/new`）

## UX 最佳实践

### 遵循的设计原则

1. **简洁主义**: 清晰的层次结构，充足的留白
2. **一致性**: 统一的卡片样式和间距
3. **反馈明确**: 所有交互都有视觉反馈
4. **响应式**: 适配桌面、平板、移动设备
5. **可访问性**: 语义化 HTML，键盘导航支持

### 避免的反模式

- ❌ 不使用 emoji 作为图标（使用 SVG 图标库）
- ❌ 不使用会导致布局偏移的 hover 效果（如 scale）
- ❌ 不使用过慢的动画（保持在 200-300ms）
- ❌ 不在浅色模式下使用低对比度文本

## 响应式设计

### 断点

- **移动端** (< 768px): 单列布局
- **平板** (768px - 1024px): 双列布局
- **桌面** (> 1024px): 三列布局

### Grid 配置

```css
grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6
```
