# AI 生成脚本大概 - 健康疾病内容引导话术

**日期**: 2026-01-28
**类型**: Feature Enhancement
**功能**: AI 生成脚本大概（Synopsis）

## 需求背景

在短视频营销中，特别是健康、医疗相关的内容，常见的做法是：
1. 分享个人或他人的真实经历
2. 展示问题得到解决或改善
3. 在结尾引导观众咨询，建立信任和转化

**用户需求**：
- 如果生成的内容与健康、疾病相关
- 在结尾自动添加引导咨询的话术
- 话术要自然、真诚，不能生硬

## 实施方案

### 修改文件

**文件**: `web/src/app/api/projects/[id]/scripts/generate-synopsis/route.ts`

### 提示词增强

在原有的提示词基础上，增加了"特别要求（健康疾病相关内容）"部分：

```typescript
特别要求（健康疾病相关内容）：
- 如果内容涉及健康、疾病、医疗、养生、治疗等主题，必须在结尾添加引导咨询的话术
- 引导话术要自然融入故事，体现真诚帮助的态度
- 引导话术参考格式：
  * "XX的病终于稳定下来了，如果有同样问题的朋友还有这种困扰的话可以找我咨询，只为帮助每一位朋友。"
  * "通过这个方法，XX的身体逐渐好转，如果你也有类似的困扰，欢迎找我交流，希望能帮到更多人。"
  * "看到XX恢复健康，我真的很欣慰，如果你身边也有朋友遇到这样的问题，可以来找我聊聊，我会尽力帮助。"
- 话术要点：
  * 先描述问题得到解决或改善
  * 表达愿意帮助他人的真诚态度
  * 使用"咨询"、"交流"、"聊聊"等温和的词汇
  * 避免过于商业化的表达
```

## 工作原理

### 1. AI 智能判断

AI 会根据项目主题、角色设定等信息，自动判断内容是否与健康疾病相关：

**触发关键词**：
- 健康、疾病、医疗
- 养生、治疗、康复
- 病症、症状、疼痛
- 中医、西医、药物
- 等等

### 2. 自动添加引导话术

如果判断为健康疾病相关内容，AI 会在剧本大概的结尾自然地添加引导话术。

### 3. 话术风格

**核心原则**：
- ✅ 真诚帮助的态度
- ✅ 温和的表达方式
- ✅ 自然融入故事
- ❌ 避免过于商业化
- ❌ 避免生硬的推销

## 话术变体

AI 会根据具体情况生成不同的话术变体，保持内容的多样性：

### 变体 1：强调问题解决
```
XX的病终于稳定下来了，如果有同样问题的朋友还有这种困扰的话
可以找我咨询，只为帮助每一位朋友。
```

### 变体 2：强调经验分享
```
通过这个方法，XX的身体逐渐好转，如果你也有类似的困扰，
欢迎找我交流，希望能帮到更多人。
```

### 变体 3：强调真诚帮助
```
看到XX恢复健康，我真的很欣慰，如果你身边也有朋友遇到这样的
问题，可以来找我聊聊，我会尽力帮助。
```

### 变体 4：强调专业经验
```
经过这段时间的调理，XX的情况明显改善了，我在这方面积累了
一些经验，如果你也需要帮助，随时可以找我交流。
```

## 技术实现

### AI 判断逻辑

AI 通过以下信息判断是否为健康疾病相关：

1. **项目主题** (`project.topic`)
2. **主题名称** (`project.themeName`)
3. **主题描述** (`project.themeDesc`)
4. **角色设定** (`characters[].description`)

### 提示词设计原则

1. **明确触发条件**: 列出健康、疾病、医疗等关键主题
2. **提供多个示例**: 让 AI 理解话术的多样性
3. **强调自然融入**: 避免生硬的添加
4. **规定话术要点**: 确保话术的质量和风格

## 用户体验

### 之前
- 用户需要手动在剧本大概中添加引导话术
- 容易忘记添加或添加不自然
- 话术风格不统一

### 现在
- AI 自动判断并添加引导话术
- 话术自然融入故事
- 多种话术变体，保持内容多样性
- 用户可以直接使用或微调

## 注意事项

### 1. 合规性

引导话术应该：
- ✅ 表达真诚帮助的态度
- ✅ 使用温和的表达方式
- ❌ 避免夸大疗效
- ❌ 避免承诺治愈
- ❌ 避免违反广告法

### 2. 真实性

- 内容应基于真实经历或案例
- 避免虚假宣传
- 尊重医疗专业性

### 3. 用户可编辑

- AI 生成的话术只是建议
- 用户可以根据实际情况修改
- 用户可以选择不使用引导话术
